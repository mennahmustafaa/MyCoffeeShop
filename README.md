# MyCoffeeShop

A SwiftUI-based iOS application for a coffee shop, designed to provide users with a seamless experience to browse, order, and manage their favorite coffee drinks.

---

## Features

- Browse coffee menu with images and descriptions
- Add items to cart and manage orders
- Favorite your preferred coffee drinks
- Authentication with sign-up, login, and password management
- Notifications for order updates
- Smooth animations and a clean, user-friendly UI
- Supports multiple coffee types, including cappuccino, latte, mocha, matcha, and more

---

## Screens

- Home Screen: Browse menu, search, PromoCode and pick locations
- Coffee Details: Detailed view for each coffee item
- Cart Screen: View and manage items in your cart
- Favorites Screen: View and manage favorite items
- Authentication Screens: Login, Sign Up, Forgot Password, Change Password
- Pickup Success Screen: Order confirmation
- Notification Screen

---

## Technologies

- SwiftUI
- Combine
- MVVM Architecture
- Supabase for backend/authentication
- Xcode 14+
